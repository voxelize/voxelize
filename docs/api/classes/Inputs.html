Class: Inputs<T>

A built-in key-bind manager for Voxelize. Uses the mousetrap library internally.

Example​

Print "Hello world" on p presses:

client.inputs.bind(
  "p",
  () => {
    console.log("Hello world");
  },
  "*"
);
