# [Voxelize](https://shaoruu.io)

A multiplayer, *super fast* voxel engine in your browser.

[Demo](https://shaoruu.io) · [Discord](https://discord.gg/9483RZtWVU) · [Docs](https://docs.voxelize.io)

</div>

## Quick Start

```bash
git clone https://github.com/shaoruu/voxelize.git && cd voxelize
pnpm install && pnpm run proto && pnpm run build
pnpm run demo  # visit http://localhost:3000
```

**Requirements:** [Rust](https://www.rust-lang.org/tools/install), [Node.js](https://nodejs.org), [cargo-watch](https://crates.io/crates/cargo-watch), [protoc](https://grpc.io/docs/protoc-installation/)
